Banking System Simulation (Java)

📌 Overview
A console-based banking system built using Java that simulates core banking operations such as account creation, deposits, withdrawals, transaction tracking, and data persistence.

🚀 Features

* Create new bank accounts
* Deposit and withdraw money
* Transfer funds between accounts
* View transaction history with timestamps
* Persistent storage using file handling
* Menu-driven CLI interface


🛠️ Technologies Used

* Java (Core)
* OOP Concepts (Encapsulation, Abstraction)
* Exception Handling
* File Handling (Serialization)
* Collections Framework (HashMap, ArrayList)


⚙️ System Design

* Accounts stored using HashMap for efficient lookup
* Each account maintains its own transaction history
* Custom exceptions used for handling insufficient balance


📊 Performance & Metrics

* O(1) average time complexity for account lookup
* Scalable design (limited only by system memory)
* Persistent storage ensures data is retained after program restart
* Transaction logging system for traceability

  
▶️ How to Run

1. Compile all Java files
2. Run the Main class
3. Use menu options to interact with the system


📈 Future Improvements

* Spring Boot integration

* Authentication system (PIN-based login)
